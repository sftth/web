​넥사크로 패키지 다국어처리 가이드

 

1. 기준언어 결정

​ - 화면 개발시에 사용할 기준언어 결정

    화면개발을 한글, 영문 또는 기타언어로 할 것인지를 결정한다.

 - 다국어 치환시에는 기준언어는 치환하지 않는다.

    Default : 한글(KR)

 - 기준언어가 변경되면 Comm.xjs에서 아래 설정을 변경한다.

 

​

​

2. 선택한 언어 데이터 확보

로그인시 선택한 언어코드에 맞추어 단어 및 메시지 데이터를 확보한다.

1. gds_language : 단어 처리용 Dataset

2. gds_message : 메시지 처리용 Dataset

​

​

​

3. gds_language Layout

 Columns

 Description

 WORD_CD

 단어코드

 WORD_TRAN

 치환될 단어

 LANG_CD

 언어코드

 

사용예)

 WORD_CD 

 WORD_TRAN 

 LANG_CD 

 저장

Save

 EN

 조회

Search

 EN

​​

​기준언어가(화면에서 사용한 언어) 한글일 경우 WORD_CD는 한글이 될 수도 있다.

 this.com_fromInit(obj);

위 함수만 호출되어 있으면 특별히 설정할 필요는 없다.
