넥사크포 플랫폼 제품패치방법

 

1) 투비소프트 기술지원 홈페이지를 접속합니다. (url : support.tobesoft.co.kr)

해당 사이트에서 Downloads->nexacro platform을 클릭하면 최신버전을 받을 수 있습니다.
 
 

 

 

 

​

2) Files 구성 설명

 

Files 

 Description 

 nexacro14_Launcher.cab

 웹 브라우저에서 배포 시 사용되는 파일로써, nexacro platform 전용브라우저를 실행시키기 위한 런처 파일입니다. 

 nexacro14_Library.zip

 nexacro library로써 안에는 .js파일이 여러 개 들어가 있습니다. HTML5, runtime 모두 해당 library를 사용합니다.

 nexacro14_SetupDeveloper.exe

 개발 시 필요한 nexacro studio 설치파일입니다. 

 nexacro14_SetupEngine.cab

 웹 브라우저에서 nexacro platform 엔진모듈을 설치하기 위한 cab파일입니다. 

 nexacro14_SetupEngine.exe

 exe파일로 전용브라우저 아이콘을 이용하여 실행시 또는 수동설치 시 엔진 업데이트 증에 사용하는 파일입니다. 

 nexacro14_VersionInfo_20150805_2.txt 

 nexacro platform에 대한 CLSID 및 버전정보가 기술되어 있습니다. 

 

​

3) 패치 적용하기 (HTML5)

​

3-1. nexacro studio 패치 적용하기

nexacro studio는 위의 파일다운로드 위치에서 nexacro14_SetupDeveloper.exe를 다운로드 받아 실행하면 자동으로 업데이트가 진행됩니다.

3-2. nexacro14lib 패치 적용하기

nexacro14_Library.zip 파일을 압축 해제 후 서버에 위치한 nexacro14lib 폴더에 덮어쓰기 합니다.

3-3. 소스 Generate 작업하기

nexacro Studio 과 lib파일 덮어쓰기가 끝난 후 기존 프로젝트 소스를 Open하여 다시 Generate작업을 수행합니다.

 

 

4) 패치 적용하기 (RUNTIME)

nexacro studio 설치파일을 설치하면 별도 작업 없이 패치가 이루어 집니다
